# **App Name**: Siya Demolitions

## Core Features:

- Home Page: Display basic business information, services, contact details, and service areas.
- Services Page: List the types of demolition services offered by Siya Demolitions Projects.
- Contact Page: Display contact information, including phone number (078 429 2760) and a contact form.
- Service Areas Page: List all the service areas in Gauteng, South Africa.
- Johannesburg Service Area Page: Specific information about services offered in Johannesburg.

## Style Guidelines:

- Primary color: Deep concrete gray (#63696D) to represent demolition expertise.
- Background color: Light gray (#E0E3E6), providing a clean and professional backdrop.
- Accent color: Burnt orange (#D97706), drawing the user's eye and conveying energy.
- Body and headline font: 'Inter' sans-serif font for a modern and clean look.
- Use a grid-based layout for clear content organization.
- Use minimalist icons for navigation and service descriptions.
- Subtle transitions when navigating between pages.
